## **Machine Learning (ML)** 
-Is a branch of artificial intelligence (AI) that enables computers to learn from data and improve their performance automatically without being explicitly programmed for every task.

### Simple Explanation
Instead of telling a computer exactly what to do step-by-step, you give it data, and it figures out patterns on its own.
- Example:
If you show a computer many pictures of cats and dogs, it can learn how to tell the difference between them.

## How It Works
1. Input Data – You provide examples (e.g., images, numbers, text)
2. Training – The computer finds patterns in the data
3. Model Creation – It builds a system (model) based on those patterns
4. Prediction/Decision – It uses the model to make decisions on new data

## Types of Machine Learning
1 *Supervised Learning*
-Learns from labeled data (answers are given)

Example: Predicting house prices

2 *Unsupervised Learning*
-Finds patterns in unlabeled data

Example: Grouping customers based on behavior

3 *Reinforcement Learning*
-Learns through trial and error with rewards or penalties

Example: Training a game-playing AI
### Real-Life Examples
-Spam filters in email

-Recommendation systems (like Netflix or YouTube)

-Voice assistants (Siri, Google Assistant)

-Fraud detection in banking (relevant to your chama/finance apps)
